PANTEÃO - Jogo de Plataforma

INSTRUÇÕES:
- Use SETAS ESQUERDA e DIREITA para mover
- ESPAÇO para pular
- J para atacar com chicote
- K para defender com escudo
- L para usar habilidade especial

ESTRUTURA:
- main.py: Arquivo principal do jogo
- sounds/: Pasta com arquivos de áudio
  - jump.wav: Som do pulo
  - attack.wav: Som do ataque
  - shield.wav: Som do escudo
  - hurt.wav: Som de dano
  - enemy_hurt.wav: Som de inimigo atingido
  - collect.wav: Som de coleta
  - ability.wav: Som de habilidade
  - door.wav: Som de porta
  - select.wav: Som de seleção
  - music.mp3: Música de fundo

EXECUÇÃO:
Execute o comando: python main.py
